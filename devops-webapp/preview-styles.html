Aperçu des Styles CSS

Deux designs professionnels au choix pour votre Dashboard DevOps

Style Default

Design moderne et dynamique avec animations fluides

Caractéristiques

  • Dégradés colorés prononcés
  • Animations dynamiques au survol
  • Ombres portées marquées
  • Design énergique et moderne
  • Parfait pour démonstrations

Utilisation Recommandée

Idéal pour les présentations dynamiques, démonstrations publiques, portfolios créatifs. Style parfait pour attirer l'attention et montrer le côté innovant de votre projet DevOps.

Style Professional

Design épuré inspiré des dashboards d'entreprise

Caractéristiques

  • Variables CSS personnalisables
  • Système de design cohérent
  • Animations subtiles
  • Style corporate épuré
  • Optimisé pour production

Utilisation Recommandée

Parfait pour environnements corporate, applications d'entreprise, présentations formelles. Design professionnel qui inspire confiance et sérieux dans un contexte business.

Comment Changer de Style

  1. Naviguez dans le dossier src :
    cd devops-app/src
  2. Sauvegardez le style actuel :
    mv App.css App-Default.css
  3. Activez le style professionnel :
    mv App-Professional.css App.css
  4. Relancez l'application :
    npm start
  5. Admirez le nouveau design !

Personnalisation

Les deux styles sont entièrement personnalisables. Consultez le fichier GUIDE_STYLES.md pour des instructions détaillées sur la personnalisation des couleurs, espacements, et autres propriétés visuelles.

Compatibilité

Les deux thèmes sont entièrement responsive et compatibles avec tous les navigateurs modernes (Chrome, Firefox, Safari, Edge). Testés sur desktop, tablet, et mobile.
